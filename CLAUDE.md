# CLAUDE.md

This project is a comprehensive toolkit for coding in Claude Code. It includes a set of plugins for various development tasks.

## Directory Structure

```
claude-code-toolkit/
├── code-review/           # Code review plugin
├── fullstack-dev/         # Full-stack development plugin
├── scripts/               # Utility scripts
├── DESIGN_PRINCIPLES.md   # Design principles
└── README.md              # README
```

## Update guidelines

- When updating plugins, update the version in the `plugin.json` file in each plugin directory and `.claude-plugin/marketplace.json` file in the root directory.
- When updating plugins, update the `README.md` files to be up-to-date with the latest features and usage.
- When updating files, update the `CLAUDE.md` to be up-to-date with the latest directory structure.
